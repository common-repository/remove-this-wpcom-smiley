== Description ==

Remove this WPcom Smiley plugin removes the wordpress.com stats smiley for all WordPress installations with the plugin JetPack.

== Installation ==

1. Download the plugin .zip file
2. Go to Plugins/Add new/Upload in the WordPress administration panel.
3. Find and select the plugin .zip file and click upload.
4. Click on "Activate Plugin"
5. Profit...?

== Frequently Asked Questions ==

= Why this plugin? =

This plugin removes the WordPress.com smiley in a safer way than most plugins and dosent require any CSS changes!
